Mail Shots

This tool allows you to create CSV (Comma Seperated Value) file which can be used by many word processors (including OpenOffice which is FREE) to create mail shots. To create a CSV file, choose Mail Shot from the Client menu.

Vaccinations

Simply use the filters on the right to choose the species, vaccines and dates that you are interested in and click the Go button. Now if you click on the Generate CSV File button you will be prompted to give the output file a name and save it disk. You can then use this file with your favourite word processor to create the mail shot.
