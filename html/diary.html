The Diary

To help incourage a paperless (or slightly less paper) office, Evette has a built in diary. You can use this tool to quickly leave yourself, a colleague or everyone a message or reminder. I thought of a number of different ways to handle the diary and opted for a slightly less logical but more user friendly system. It is easiest to explain with examples.

Creating Diary Notes

The quickest way to create a diary note is to choose Add to Diary from the Diary menu. This will give you this panel:

Here you can specify:

Date
The date that you would like the note to become active
Username and Position
This is a little complicated. If you would like your diary note to be for the attention of everybody, leave these fields blank. If you would like this note to be for the attention of a particular user enter the username here (or choose it from the associated drop down). If you would like the note to be for the attention of a particular group of staff, for example managers, vets, receptionists etc.. then enter their position here (or choose it from the associated drop down).
Subject
Enter a (very) brief description of the notes contents here.
Note
This is where you can write the body of your note.
Removed
Leave this blank. When you want to remove the note from the diary, then put todays date in here.

When you are happy with your note, click on the Submit button at the bottom.

Viewing and Editing Diary Notes

You can view the diary by choosing Edit Diary from the Diary menu or by clicking on the Edit Diary tool bar button. This will bring up this panel:

At the top of this panel you will find a set of filters that you can use to adjust which diary notes are shown. If you leave them as they are, you will be shown all active diary notes addressed to you personally or addressed to your position or addressed to everybody. This is exactly what you are most likely to want to see. If however you wanted to view removed notes, notes addressed to other users or other positions or notes not yet active, you can adjust these filters to suit your needs.

The list of notes is shown in the large panel on the left. If you click on an item in this list the entire note will be displayed in the large panel to the right. To edit a note, select the note you would like to edit and click on the Edit button or just double click on the note. To delete a note, select the note you would like to delete and click on the Delete button (Note: this will permanently delete the note. If you want to mark a note as completed or removed, edit the note and put todays date in the Removed field. This will allow you to retrieve the note or check when it was removed and by who should you need to).

Linking Diary Notes to Records

To save time. If you want to leave a note regarding a particular client, animal or appointment you can link the note to the record. This way when the note is viewed, the associated record is just a click away. To create a linked diary note. Open the client, animal or appointment that you want to link to, and click on the Create Diary Note button:

Then fill in the note as normal. When viewed the note have a button which will take you to the associated record.
