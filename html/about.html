  • Add support for postgresql - this has turned out to be pretty complex, mysql is much more flexible when it comes to allowing evette to handle the creation and setting up of the evette database. Postgres will not allow this, it should be possible to set up evette to use postgres but much of the setup will be down to the user. I have had a tinker with this but I have a lot more to learn about postgres yet. Please let me know if this a feature that is important to you.
